We can use a Repeat to make drawing shapes a lot easier! You can see below a script to draw a square.



Using a Repeat try to draw the following regular* shapes:
  • Equilateral Triangle
  • Pentagon
  • Hexagon
  • Octagon
  • Circle
  • Five-sided Star
Save the scripts as you go because next you'll be trying to figure out a pattern.

* - Regular means all the sides are the same length and all the turning angles are the same.
