Now, you are going to make a block that takes two inputs. We want to create a draw shape block that takes a number of sides and a number of pixels for the length of each side. We will call these input arguments n and pixel. This exercise should be done with a partner, so introduce yourself to your neighbor and get started on the exercise together!

By the way, you can create the inputs to this block in exactly the same way as we did in the previous section, by clicking on the plus signs to add input; however, as a shortcut, you can also type the names of the input as shown below.


The percent signs (%) indicate that the word should be an input. We want you to feel comfortable with both entry methods.

Unfortunately, the size of the text field where you can type in the name of the block (and any %input shortcuts) is limited. If you find yourself running out of room, we recommend that you just name the block and use the plus signs later to add inputs.
