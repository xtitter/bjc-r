We will now make a different kind of block -- a reporter block. To demonstrate this, we will make a block called max that takes two numbers as input and reports the bigger value (the maximum).



  1. Click Make a block and select the Operators tab. We want a reporter block. This will give the block its round shape as shown above. As the name implies, reporter blocks can report a value. In the image below, you can see that we used the % shortcut for making input variables.


  2. This should give you a blank Block editor. Notice the words report: something. You can either type where it says something, or you can drag other blocks there. We need to figure out what should be reported.



    3. To keep track of the value to be reported, we are going to make another variable:

    Use a Script Variables block. You can click on the name of the variable and change it to bigger value.



    We can use an if/else block to set the variable bigger value to either x or y.
    Drag a set block into the Block Editor from the regular variables menu.
    You can click on the down arrow on the set block to select bigger value.
    Then, you will drag the bigger value variable into the box next to report.
    Now, click OK and try the block out!
